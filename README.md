# docker-ghost (Archived)

> **This repository is archived and no longer maintained.**

When this image was created, the official Ghost Docker image lacked production-ready configuration. That is no longer the case.

## Use the official image instead

The official [Ghost image on Docker Hub](https://hub.docker.com/_/ghost) is well-maintained, up to date, and covers all use cases including production deployments.

```
docker pull ghost:latest
```

For production setups, refer to the [official Ghost Docker documentation](https://hub.docker.com/_/ghost).
